# 🚀 AI Agent using Google ADK

## 📌 Project Overview
This project is my first AI Agent built using Google ADK and Gemini.  
The agent can understand user prompts, use tools like Google Search, and generate intelligent responses.

---

## 🧠 What this project does
- Creates an AI agent using Google ADK
- Uses Gemini LLM as the brain
- Uses Google Search tool for real-time information
- Runs in a web-based interface (ADK Web UI)
- Shows step-by-step reasoning (trace)

---

## ⚙️ Tools & Technologies Used
- Google ADK (Agent Development Kit)
- Gemini API (LLM)
- Python
- Kaggle Notebook

---

## 💬 Example Usage
User can ask:
- What is AI?
- Latest AI news
- Weather in different cities
- Real-world questions requiring search

---

## 📚 What I learned
- How AI agents work
- How LLM + tools combine
- How agents decide when to search vs answer directly
- How to build and test AI systems

---

## 🎯 Outcome
Successfully built and tested my first working AI agent system with real-time tool usage and reasoning trace.
